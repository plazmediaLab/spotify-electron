Clonar repositorio:

```
$ git clone https://github.com/plazmediaLab/spotify-electron.git
```

Yanr - administrador de paquetes

```
$ cd spotify-electron

$ yarn add
```

Ejecutar App

```
$ yarn electron-dev
```
